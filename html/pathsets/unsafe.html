Unsafe

The unsafe logic group is meant for paths which have not been sorted in any difficulty group. The only rule to put a path in unsafe is that the path is not following the rules of the other groups. Here is a non-exhaustive list of things that can be required in unsafe:

  • A path that requires more than one damage boost.
  • Using grenade or sentry to redirect projectiles.
  • Using bash to move enemies around.
  • Bash glide[](https://github.com/sparkle-preference/OriWotwRandomizerClient/wiki/Bash-glide).
  • Dash ramp[](https://github.com/sparkle-preference/OriWotwRandomizerClient/wiki/Dash-ramp).
  • Sentry, Flash, Spike, Blaze, Shuriken for movement.
  • Grenade jumps,
    Weapon jumps,
    Sword pogo,
    Glide jump and
    Launch boost for movement.
  • Aerial Sentry jumps
  • Grenade to go through darkness in Depths.

Unsafe logic is also removing all requirement in order to enter Woods, Pools, Depths, Baur, Wastes and Willow while Gorlek requires Regenerate and Moki requires between 4.5 and 6.5 HP in addition to Regenerate.
